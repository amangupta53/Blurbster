A simple python script to generate common blurbs.

Requirements
============
Python 3.x

Instructions
============
Copy and Paste your blurb in blurb.txt and replace the following:

i-xxxxxxxxxxx --> instanceid
vol-xxxxxxxxx --> volumeid
us-xxxxx-xx   --> az

Run modder.py in python.
